SKC - system knoledge control
система контроля знаний.
Не тестирования, хотя это одна из функций, - а именно получение знаний.
В теории должна быть нормальная читалка, видение списков чтения, тестовая система знаний.
При чём тестовая система должна отслеживать блоки для повторения и предлагать каждый день повторить какое-то знание.
Моё нынешнее видение по версиям.
0.0.1 - консольное создание списков.
0.5  - наличие графического интерфейса.
1.0 - работа с базой данных, 
      наличие читалки, 
      отслеживание по календарю повторений(интервальное повторение),
      создание списков.
2.0 - веб-страница,
      синхронизация с PC версией
